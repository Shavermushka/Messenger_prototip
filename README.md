# 📡 MessengerPro — Мессенджер в стиле Discord

Мой ооочень простенький мессенджер. Проект создан с использованием Flask и Socket.IO.

---

## 🎯 Особенности проекта

* **Чат в реальном времени** с мгновенной доставкой сообщений
* **Удобный интерфейс** в стиле Discord
* **Приватные чаты** между пользователями
* **Групповые чаты** для команд
* **Система уведомлений** о новых сообщениях
* **Авторизация** пользователей

---

## 🛠️ Технологии

* **Backend**: Python, Flask
* **WebSocket**: Socket.IO
* **Frontend**: HTML/CSS/JS
* **Версии зависимостей**:
  * Flask: 3.1.2
  * Flask-SocketIO: 5.5.1
  * Python-EngineIO: 4.12.3
  * Python-SocketIO: 5.15.0
  * Simple-WebSocket: 1.1.0

---

## 🚀 Установка и запуск

1. **Клонируйте репозиторий**:
```bash
git clone https://github.com/Shavermushka/MessengerPro.git
```

Установите зависимости:
```bash
pip install -r requirements.txt
```
Запустите сервер:
```bash
python server.py
```
Откройте в браузере:
```bash
http://localhost:5000
```

### 📊 Требования к окружению:

 - Python 3.12+
 
- pip (для установки зависимостей)

- Права на запуск сервера

🔐 Безопасность

- Все чувствительные данные хранятся локально

- Сообщения передаются через защищенный WebSocket

- Авторизация пользователей реализована через токены
